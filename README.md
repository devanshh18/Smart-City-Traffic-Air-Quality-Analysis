# Smart City Traffic & Air Quality Data Analysis

## Overview

Urban cities face increasing traffic congestion and rising air pollution, which significantly impact public health, mobility, and urban sustainability. This project performs an end-to-end data analysis of traffic and air-quality datasets to identify congestion patterns, pollution hotspots, and key urban planning insights.

The analysis combines **SQL, Python, and Power BI** to build a complete data pipeline from raw data storage to analytical insights and dashboard visualization for smart-city decision making.


---

## Tools & Technologies

- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- Jupyter Notebook

---

## Datasets

Two datasets were used for the analysis.

### Traffic Dataset
- **15,000 records**
- Includes traffic density, vehicle counts, area types, and signal waiting times
- Covers daily traffic patterns across different urban zones

### Air Quality Dataset
- **3,329 records**
- Covers **254 cities across 31 states**
- Includes pollutant measurements from **499 monitoring stations**

---

## Project Workflow

1. Data ingestion and storage in **MySQL**
2. Data cleaning and preprocessing using **Python**
3. Exploratory Data Analysis (EDA)
4. Traffic load feature engineering
5. Pollution severity classification
6. Smart-city analytical interpretation
7. Interactive **Power BI dashboard creation**
